⚖️ 化学键能量计算器

键能计算原理

键能是衡量化学键强度的物理量，表示破坏1mol化学键所需的能量：

键能计算公式
$$E_{bond} = \frac{\Delta H}{n}$$

其中：

  • $E_{bond}$：键能 (kJ/mol)
  • $\Delta H$：反应焓变 (kJ)
  • $n$：化学键数量
常见化学键键能对比
  • H-H：436 kJ/mol
  • C-C：347 kJ/mol
  • C=O：799 kJ/mol
  • N≡N：945 kJ/mol
  • Na-Cl：787 kJ/mol

🧲 分子极性实验

分子极性测试

分子极性原理

分子极性取决于分子中正负电荷中心的相对位置：

影响极性的因素
  • 键的极性：电负性差异
  • 分子几何构型：对称性
  • 孤电子对：影响电子云分布
极性的应用
  • 溶解度：相似相溶原理
  • 沸点：极性分子间作用力更强
  • 化学反应：影响反应活性
  • 生物功能：蛋白质折叠、酶催化
极性判断方法
  1. 分析分子中所有化学键的极性
  2. 确定分子几何构型
  3. 判断正负电荷中心是否重合
  4. 重合为非极性，不重合为极性

现实生活中的化学键

离子键应用 - 食盐 (NaCl)

食盐中的离子键使其具有高熔点、易溶于水的特性，广泛应用于食品调味和化工生产。

第一章：原子世界

探索构成万物的基本粒子

🧩 原子结构

原子是化学变化中的最小粒子，由原子核和核外电子构成：

  • 质子：带正电，决定元素种类
  • 中子：不带电，影响原子质量
  • 电子：带负电，决定化学性质
原子构成公式
$$原子序数 = 质子数 = 核外电子数$$
$$质量数 = 质子数 + 中子数$$

同位素概念

具有相同质子数、不同中子数的同一元素的不同原子互称为同位素。

示例：氢的三种同位素：氕(H)、氘(D)、氚(T)

原子结构交互模型

电子排布规律

K层 (最多2e⁻)
L层 (最多8e⁻)
M层 (最多18e⁻)
N层 (最多32e⁻)
填充规则：
  • 能量最低原理
  • 泡利不相容原理
  • 洪特规则

⚡ 电子排布

核外电子按照特定规则分层排布：

能量最低原理

电子优先占据能量较低的轨道

泡利不相容原理

同一轨道最多容纳2个自旋方向相反的电子

洪特规则

电子在等价轨道上尽可能分占不同轨道且自旋平行

🗺️ 元素周期表 - 化学的"地图"

周期表规律

  • 周期：电子层数相同，从左到右金属性减弱
  • 族：最外层电子数相同，化学性质相似
  • 分区：s区、p区、d区、f区

周期性变化规律

原子半径

同周期↘，同主族↗

金属性

同周期↘，同主族↗

电负性

同周期↗，同主族↘

第二章：化学键与分子间作用力

探索原子如何结合形成分子

🔗 化学键类型

化学键是原子间相互作用的力，主要有三种类型：

离子键

金属原子与非金属原子间通过电子转移形成

  • 电子从金属原子转移到非金属原子
  • 形成带正负电荷的离子
  • 通过静电作用结合
示例：NaCl（氯化钠）

共价键

非金属原子间通过电子共用形成

  • 原子间共用电子对
  • 形成稳定的电子构型
  • 可以是单键、双键或三键
示例：H₂O（水分子）

金属键

金属原子间通过电子海模型结合

  • 金属原子释放价电子形成电子海
  • 正离子沉浸在电子海中
  • 赋予金属导电性、延展性
示例：Cu（铜）

⚡ 离子键形成过程

钠与氯形成离子键

离子键形成原理

钠原子（Na）失去一个电子形成Na⁺离子，氯原子（Cl）获得一个电子形成Cl⁻离子：

Na + Cl → Na⁺ + Cl⁻ → NaCl
能量变化
  • 电离能：Na → Na⁺ + e⁻ (需要能量)
  • 电子亲和能：Cl + e⁻ → Cl⁻ (释放能量)
  • 晶格能：Na⁺ + Cl⁻ → NaCl (释放大量能量)
离子化合物性质
  • 高熔点、高沸点
  • 固态不导电，熔融态或溶液导电
  • 硬而脆
  • 易溶于极性溶剂

☁️ 共价键与电子云

共价键形成原理

原子间通过共用电子对形成稳定的电子构型：

单键

共用一对电子，如H-H

双键

共用两对电子，如O=O

三键

共用三对电子，如N≡N

电子云重叠

共价键的本质是原子轨道重叠形成分子轨道，电子云密度在原子核间区域最大。

键的极性

根据电负性差异判断：

  • 非极性共价键：相同原子间（H-H）
  • 极性共价键：不同原子间（H-Cl）

🔺 分子几何构型

VSEPR理论模型

VSEPR理论

价层电子对互斥理论：分子采取电子对相互排斥最小的空间构型。

电子对数 孤电子对数 分子构型 示例
2 0 直线形 CO₂
3 0 平面三角形 BF₃
4 0 四面体形 CH₄
4 1 三角锥形 NH₃
4 2 V形 H₂O
分子极性判断

分子极性取决于：

  • 键的极性
  • 分子几何构型
  • 对称性

示例：CO₂（非极性）vs H₂O（极性）

💧 分子间作用力

分子间作用力类型

范德华力

所有分子间都存在，包括：

  • 取向力：极性分子间
  • 诱导力：极性-非极性分子间
  • 色散力：所有分子间（主要作用）
氢键

特殊的分子间作用力：

  • 存在于H与N、O、F之间
  • 比范德华力强
  • 影响物质物理性质
示例：水分子间的氢键
对物质性质的影响
  • 沸点、熔点
  • 溶解度
  • 表面张力
  • 粘度
