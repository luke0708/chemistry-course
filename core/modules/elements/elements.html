金属元素家族

金属元素占据元素周期表的大部分区域，具有独特的物理和化学性质。

碱金属

第1族：Li, Na, K, Rb, Cs, Fr

特性：

  • 最外层1个电子，极易失去
  • 化学性质非常活泼
  • 低密度、低熔点
  • 与水剧烈反应

典型反应：

2Na + 2H₂O → 2NaOH + H₂↑

碱土金属

第2族：Be, Mg, Ca, Sr, Ba, Ra

特性：

  • 最外层2个电子
  • 化学性质活泼
  • 密度和熔点比碱金属高
  • 重要生物元素（Ca, Mg）

典型反应：

Ca + 2H₂O → Ca(OH)₂ + H₂↑

过渡金属

第3-12族：Fe, Cu, Zn, Ag, Au等

特性：

  • d轨道电子填充
  • 多种氧化态
  • 形成有色化合物
  • 良好的导电导热性

典型应用：

铁（钢铁）、铜（导线）、金（饰品）

合金：金属的"混搭"艺术

合金是由两种或多种金属（或金属与非金属）组成的具有金属特性的物质。

非金属元素

非金属元素大多位于元素周期表右上角，包括卤素、氧族、氮族等。

卤素

第17族：F, Cl, Br, I, At

特性：

  • 最外层7个电子，易得1个电子
  • 化学性质非常活泼
  • 从气体到固体，颜色逐渐加深
  • 氧化性强弱：F₂ > Cl₂ > Br₂ > I₂

典型反应系列：

Cl₂ + 2NaBr → 2NaCl + Br₂
Br₂ + 2KI → 2KBr + I₂

氧族元素

第16族：O, S, Se, Te, Po

特性：

  • 最外层6个电子
  • 从非金属到准金属过渡
  • 多种同素异形体
  • 重要生物元素（O, S）

应用：

  • 氧气：呼吸、燃烧
  • 硫磺：火药、橡胶
  • 硒：光电材料

氮磷化合物

N, P及其化合物

氮元素：

  • N≡N三键，化学性质稳定
  • 氨（NH₃）：重要化工原料
  • 硝酸（HNO₃）：强酸、氧化剂

磷元素：

  • 多种同素异形体（白磷、红磷）
  • 磷酸盐：肥料、洗涤剂
  • DNA、ATP的重要组成

非金属元素性质变化规律

同族从上到下：

  • 非金属性减弱
  • 氧化性减弱
  • 氢化物稳定性降低

同周期从左到右：

  • 非金属性增强
  • 电负性增大
  • 原子半径减小

有机反应类型
